# 🌱 RefSpring

**RefSpring** is a lightweight affiliate management system built for creators, indie hackers, and SaaS founders.  
It lets you create referral campaigns, track performance, and reward your ambassadors — without relying on expensive third-party platforms.

> 💼 Born out of real needs while launching [Oridium](https://getoridium.com), RefSpring is now built to serve others.

---

## 🚀 Features

- ✅ Create and manage referral campaigns
- ✅ Generate unique tracking links for affiliates
- ✅ Track clicks, conversions, and earnings
- ✅ Admin dashboard with campaign performance
- ✅ Public affiliate page with live stats
- ✅ Lightweight, no-framework stack (HTML/CSS/JS)
- ✅ Supabase for database & auth
- ✅ Stripe integration-ready
